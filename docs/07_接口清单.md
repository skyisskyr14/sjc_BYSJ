# 07 接口清单

## 物资/仓库/库存
- `GET /sjc/material/page`
- `POST /sjc/material/create|update|delete`
- `GET /sjc/warehouse/page`
- `POST /sjc/warehouse/create|update|delete`
- `GET /sjc/inventory/page`
- `GET /sjc/inventory/flow/page`
- `POST /sjc/inventory/inbound|outbound`

## 预警/看板
- `GET /sjc/alert/page`
- `POST /sjc/alert/ack`
- `GET /sjc/dashboard/metrics|trend`

## 调度
- `GET /sjc/dispatch/task/list`
- `POST /sjc/dispatch/task/status`
- `GET /sjc/dispatch/route`
- `POST /sjc/dispatch/track/report`
- `POST /sjc/dispatch/track/batch`
- 兼容：`GET /sjc/task/list`

## Outbox管理
- `GET /sjc/outbox/dead/page`
- `POST /sjc/outbox/retry?id=`

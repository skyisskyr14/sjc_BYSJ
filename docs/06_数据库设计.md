# 06 数据库设计

## 与 sq_system 关系
- 用户基础信息在 `sq_system.user`
- `sjc_user_role`、`sjc_user_warehouse` 通过 `sys_user_id` 关联系统用户

## 核心表
- `sjc_material`：物资字典与阈值
- `sjc_warehouse`：仓库主数据
- `sjc_inventory`：库存现状
- `sjc_inventory_flow`：出入库流水（`biz_no` 唯一）
- `sjc_alert`：预警（UNCONFIRMED/ACKED/...）
- `sjc_dispatch_task`：任务主表（DRAFT->APPROVED->RUNNING->ARRIVED->DONE/CANCELED）
- `sjc_dispatch_task_log`：状态流转日志
- `sjc_dispatch_track`：轨迹点
- `sjc_event_outbox`：事件可靠投递

## 索引策略
- 库存流水按 `create_time+flow_type+warehouse_id+material_id` 复合索引（V006）
- 轨迹按 `task_id+create_time` 索引（V006）

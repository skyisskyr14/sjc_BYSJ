# 04 验收与冒烟

## 前置
- 后端已启动：`http://localhost:10438/api`
- `db/sjc/V001~V005` 已执行

## 执行
```bash
bash scripts/smoke.sh
# 或指定地址
BASE_URL=http://localhost:10438/api bash scripts/smoke.sh
```

## 覆盖链路
1. health 检查
2. 新增仓库
3. 新增物资
4. 入库
5. 出库成功
6. 出库失败（库存不足断言）
7. dashboard 指标读取
8. 预警查询/确认
9. 调度任务状态流转
10. 轨迹点上报

## 输出
- 终端输出 `PASS/FAIL` 与关键响应摘要。

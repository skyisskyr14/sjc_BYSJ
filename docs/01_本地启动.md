# 01 本地启动
1. 执行 SQL：`db/sjc/V001 -> V002 -> V003 -> V004 -> V005`
2. 启动中间件：`docker compose -f deploy/docker-compose.yml up -d`
3. 启动后端：`cd spring && mvn -pl dev-admin spring-boot:run`
4. 启动前端：`cd web && npm run serve`

5. ClickHouse 初始化（可选）：`clickhouse-client --multiquery < db/ch/V001__inventory_history.sql`
6. 注意：`db/ch/*` 是 ClickHouse 语法，勿在 MySQL 中执行。
